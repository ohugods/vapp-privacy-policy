Política de Privacidade

Aplicativo VAPP - Vitória Ambiental

Sobre esta política

Este documento descreve como o VAPP trata os dados dos seus usuários.

Resumo da nossa política

Este aplicativo é destinado exclusivamente para uso interno da Vitória Ambiental.

Nenhum dado pessoal é coletado, armazenado ou compartilhado com terceiros.

O aplicativo opera apenas dentro do ambiente corporativo da empresa, com dados internos e de acesso restrito.

Coleta e uso de informações

O VAPP coleta apenas os dados necessários para a operação interna da empresa, incluindo:

  • Dados de login para autenticação de funcionários
  • Registros de operações internas
  • Informações de rastreamento para fins operacionais

Segurança

Todos os dados são armazenados de forma segura em nossos servidores internos e/ou em serviços de nuvem com proteções adequadas.

O acesso é restrito apenas a funcionários autorizados da Vitória Ambiental.
